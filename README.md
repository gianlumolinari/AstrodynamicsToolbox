# astroToolbox

A MATLAB astrodynamics toolbox for research and mission design.

## Planned modules
- Classical astrodynamics
- Lambert solvers
- Optimization
- CR3BP
- Periodic orbits
- Invariant manifolds

## Structure
- `+astro/` library code
- `examples/` demos
- `tests/` validation scripts
- `docs/` notes
- `dev/` experimental work

## Usage
Run `startup.m` before using the toolbox.
